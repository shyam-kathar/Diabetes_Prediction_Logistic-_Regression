About the Project
The goal of this project is to create a predictive system that can identify the presence of diabetes in patients with high accuracy. This is a Binary Classification problem where the outcome is either 1 (Diabetic) or 0 (Non-diabetic).

# Tech Stack
Language: Python

Libraries: * Pandas (Data manipulation)

NumPy (Numerical operations)

Matplotlib / Seaborn (Data visualization)

Scikit-learn (Model training and evaluation)

# Project Workflow
Data Preprocessing: Handled missing values, performed feature scaling, and analyzed correlations.

Exploratory Data Analysis (EDA): Visualized the distribution of features and the balance of the target classes.

Model Training: Split the data into training (80%) and testing (20%) sets and applied the Logistic Regression algorithm.

Prediction: Used the trained model to predict outcomes on unseen test data.

Evaluation: Measured performance using a Confusion Matrix, Accuracy Score, and Classification Report.

# Evaluation Results
The model achieved the following performance metrics:

Accuracy: 75%

Precision/Recall: High performance in identifying : Non-diabetic .

Confusion Matrix: Clearly shows the True Positives and True Negatives.
